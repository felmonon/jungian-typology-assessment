# Website: https://typejung.com

================================================================================
ABOUT
================================================================================

TypeJung is a free psychological assessment tool based on Carl Jung's
original theory of psychological types (1921) and the Singer-Loomis methodology.

Unlike simplified 4-letter type indicators (MBTI), this assessment measures all
8 cognitive functions independently on a spectrum, providing a unique profile
rather than forcing users into one of 16 predetermined boxes.

================================================================================
THE 8 COGNITIVE FUNCTIONS
================================================================================

PERCEIVING FUNCTIONS:
- Ni (Introverted Intuition): Pattern recognition, future visioning, convergent insights
  Learn more: https://typejung.com/functions/ni

- Ne (Extraverted Intuition): Possibilities, brainstorming, divergent thinking
  Learn more: https://typejung.com/functions/ne

- Si (Introverted Sensing): Memory, tradition, detailed recall
  Learn more: https://typejung.com/functions/si

- Se (Extraverted Sensing): Present-moment awareness, action, sensory engagement
  Learn more: https://typejung.com/functions/se

JUDGING FUNCTIONS:
- Ti (Introverted Thinking): Internal logic, analysis, framework building
  Learn more: https://typejung.com/functions/ti

- Te (Extraverted Thinking): External organization, efficiency, measurable results
  Learn more: https://typejung.com/functions/te

- Fi (Introverted Feeling): Personal values, authenticity, moral compass
  Learn more: https://typejung.com/functions/fi

- Fe (Extraverted Feeling): Social harmony, emotional attunement, group dynamics
  Learn more: https://typejung.com/functions/fe
